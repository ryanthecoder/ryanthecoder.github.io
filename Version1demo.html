Demo

This was a demo I created to demonstrate the concept of my project while working on the software to make it fully functional. The electromagnet was not functioning yet at the time of filming and the object being printed was chosen in order to have a speedy demo. I hope to have more demos coming soon, showing the electromagnet working and creating a useful object.

The code written for this demo runs on the Raspberry Pi and communicates with the makerbot and the crane over TTY sockets. A python script sends the x3g code to the Replicater while it listens on the update percentage command. When it hits 4 percent it pauses the print and runs the gcode file that I wrote for the crane, resumeing when this is complete. To view the code visit here.
